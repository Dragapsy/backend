# StorynGo Backend

## Demarrer

- Lancer l'application :

```powershell
& "C:\Users\Aymane\Desktop\DEV\Projet annuelle\backend\mvnw.cmd" spring-boot:run
```

- Lancer les tests :

```powershell
& "C:\Users\Aymane\Desktop\DEV\Projet annuelle\backend\mvnw.cmd" test
```
